Image Tagging using Tensor FLow

Identifying Relevent Tags In An Image

Following Python code can be used to get tags relevent from an image 
  import tensorflow as tf
  import numpy as np
  from tensorflow.keras.preprocessing import image

  from tensorflow.keras.applications import  InceptionV3
  from tensorflow.keras.applications.inception_v3 import preprocess_input, decode_predictions

  def get_predicted_tag(file_name_with_path):

      model = InceptionV3(weights='imagenet')

      img = image.load_img(file_name_with_path, target_size=(299, 299))
      x = image.img_to_array(img)
      x = np.expand_dims(x, axis=0)
      x = preprocess_input(x)

      preds = model.predict(x)
      return decode_predictions(preds, top=5)[0]

  if __name__ =="__main__":
      tags=get_predicted_tag('Screenshot from 2020-05-16 09-37-37.png')
      print(tags)

This code was tested on python 3.8. Make sure numpy and tensor flow is already installed
